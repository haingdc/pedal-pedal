Đây không bài viết nơi tôi để lại một vài câu lệnh đôi khi cần tới mà lại quên. Nó sẽ chia sẻ không gian với các bài viết bình thường khác trong blog.

## Npm view để xem các versions của package

Kết quả cho ra các version kể cả version pre-release:

```terminal
npm view <package name> versions
```
